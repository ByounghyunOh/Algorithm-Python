**List Slicing**
```python
# list[start:End(index-1):Step]
a = [10,20,30,40]
>> a[1:3] # index 1,2 (not include 3, for (i=0: i<3: i++))
[20, 30]
>> a[::-1] # Reverse (index -1 is from last of List)
[40,30,20,10]
```

**List Comprehension**

```python
>>> a = []
>>> for n in range(1, 10+1):
        if n % 2 == 1:
            a.append(n * 2)
[2, 6, 10, 14, 18]

# Same as above (comprehension)
>>> [n * 2 for in range(1, 10+1) if n % 2 == 1]
[2, 6, 10, 14, 18]
```

### Loop
``` Python
# Sample 1
text = 'test'
for c in text:
    print(c)

# Sample 2
list = [1,3,5,7,9]
for i in list:
    print(i)


# Sample 3
# range([start,]stop[,step])
for x in range(6): # 0 to 5, 6 is not included

# Sample 4
for x in range(2, 30, 3): # 2 to 29, ++3
    if x <= 10:
        continue  # if x <= 10, continue back to loop
    else
        print(x)

# Sample 5
i = 0
while (i < 6):
    print(x)
    if i == 4:
        break
    i += 1

```
